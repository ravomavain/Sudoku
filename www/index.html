Sudoku Solver using Factor Graphs

This project is a simple sudoku solver demonstrating the use of Factor Graphs and the Sum Product Algorithm.

It was developped in Python by Romain Labolle as part of an internship at CTTC labs in Barcelona.

You can find more informations about Factor Graphs and the Sum Product Algorithm used in this project in my internship report

Source Code

The source code of this project and everything related to it can be found on github.

It requires Python 3 and Numpy for the sudoku solver and Gevent for the telnet server.

The repository also includes the LaTeX sources of the report which contains a helper function to easily generate nice Factor Graphs.

Telnet Demo

This server is also running a live demo of the solver via telnet on port 23. To access it, simply run the command telnet sudoku.cat.

The source code of this server, based on telnetsrvlib, is also included in the source code repository.
